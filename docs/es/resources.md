# Recursos

Una colección de recursos oficiales y de la comunidad para profundizar en Vue.js más allá de la preparación de entrevistas.

## Documentación oficial

- [Vue.js](https://vuejs.org/) - La documentación del framework. Empieza aquí.
- [Vue Router](https://router.vuejs.org/) - Librería oficial de routing.
- [Pinia](https://pinia.vuejs.org/) - Gestión de estado oficial.
- [Nuxt](https://nuxt.com/) - Framework full-stack con SSR, rutas basadas en archivos y auto-imports.
- [VitePress](https://vitepress.dev/) - Generador de sitios estáticos con Vite y Vue (lo que usa esta web).
- [Vite](https://vite.dev/) - Herramienta de build y servidor de desarrollo.
- [Vitest](https://vitest.dev/) - Framework de testing unitario para proyectos Vite.
- [Vue Test Utils](https://test-utils.vuejs.org/) - Utilidades oficiales de testing para componentes Vue.
- [Vue DevTools](https://devtools.vuejs.org/) - Extensión de navegador para depurar aplicaciones Vue.

## Librerías de la comunidad

- [VueUse](https://vueuse.org/) - Colección de utilidades de composición por Anthony Fu.
- [Pinia Colada](https://pinia-colada.esm.dev/) - Gestión de estado asíncrono (data fetching, caché) para Pinia.
- [VueFire](https://vuefire.vuejs.org/) - Bindings de Firebase para Vue.
- [Formkit](https://formkit.com/) - Framework de formularios para Vue.
- [PrimeVue](https://primevue.org/) - Librería de componentes UI.
- [Radix Vue](https://www.radix-vue.com/) - Componentes accesibles sin estilos.
- [Nuxt UI](https://ui.nuxt.com/) - Librería UI del equipo de Nuxt.

## Aprendizaje

- [Vue School](https://vueschool.io/) - Cursos en vídeo de principiante a avanzado.
- [Vue Mastery](https://www.vuemastery.com/) - Cursos en vídeo, muchos con miembros del core team.
- [Vue.js: The Documentary](https://www.youtube.com/watch?v=OrxmtDw4pVI) - La historia de Vue contada por Evan You.
