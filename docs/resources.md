# Resources

A curated collection of official and community resources to deepen your Vue.js knowledge beyond interview prep.

## Official documentation

- [Vue.js](https://vuejs.org/) - The core framework docs. Start here.
- [Vue Router](https://router.vuejs.org/) - Official routing library.
- [Pinia](https://pinia.vuejs.org/) - Official state management.
- [Nuxt](https://nuxt.com/) - Full-stack Vue framework with SSR, file-based routing, and auto-imports.
- [VitePress](https://vitepress.dev/) - Static site generator powered by Vite and Vue (what this site runs on).
- [Vite](https://vite.dev/) - Build tool and dev server.
- [Vitest](https://vitest.dev/) - Unit testing framework for Vite projects.
- [Vue Test Utils](https://test-utils.vuejs.org/) - Official testing utilities for Vue components.
- [Vue DevTools](https://devtools.vuejs.org/) - Browser extension for debugging Vue apps.

## Community libraries

- [VueUse](https://vueuse.org/) - Collection of composition utilities by Anthony Fu.
- [Pinia Colada](https://pinia-colada.esm.dev/) - Async state management (data fetching, caching) for Pinia.
- [VueFire](https://vuefire.vuejs.org/) - Firebase bindings for Vue.
- [Formkit](https://formkit.com/) - Form building framework for Vue.
- [PrimeVue](https://primevue.org/) - UI component library.
- [Radix Vue](https://www.radix-vue.com/) - Unstyled, accessible components.
- [Nuxt UI](https://ui.nuxt.com/) - UI library by the Nuxt team.

## Learning

- [Vue School](https://vueschool.io/) - Video courses from beginner to advanced.
- [Vue Mastery](https://www.vuemastery.com/) - Video courses, many featuring core team members.
- [Vue.js: The Documentary](https://www.youtube.com/watch?v=OrxmtDw4pVI) - The story of Vue told by Evan You.

## Blogs and articles

- [The Vue Point](https://blog.vuejs.org/) - Official Vue.js blog.
- [Evan You](https://evanyou.me/) - Creator of Vue, Vite, and VitePress.
- [Anthony Fu](https://antfu.me/) - Vue/Vite core team. Author of VueUse, Slidev, UnoCSS.
- [Daniel Roe](https://roe.dev/) - Nuxt lead.
- [Eduardo San Martin Morote](https://esm.dev/) - Vue Router and Pinia maintainer.

## Community

- [Vue Land Discord](https://chat.vuejs.org/) - Official Vue.js Discord server.
- [VueConf](https://vueconf.us/) - Vue.js conference.
- [Vue.js Amsterdam](https://vuejs.amsterdam/) - Largest Vue conference in Europe.
- [r/vuejs](https://www.reddit.com/r/vuejs/) - Vue subreddit.

## Podcasts

- [DejaVue](https://www.yourpodcast.page/dejavue) - Podcast by Alexander Lichter and Michael Thiessen.
